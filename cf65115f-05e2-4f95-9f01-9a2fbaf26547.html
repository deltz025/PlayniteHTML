Description

Last updated November 9th 2020

Created in one month for the New Jam City 2017 game jam hosted by https://waypoint.vice.com/en_us


Get the soundtrack here: https://aheadofitstime.bandcamp.com/album/yo-noid-2-enter-the-void-ost

Please visit us at noid.pizza!


Controller recommended.  Input bindings:

Left Stick/D-Pad/WASD Keys/Arrow Keys - Move Player

Right Stick/Mouse - Move Camera

A/Space/Z Key - Jump

X/B/Right Bumper/Left Click - Tether

Y/E Key/C Key/Return Key - Interact

Right Thumb/Tab - Auto Cam Toggle

Start/Escape - Pause
